About Sorting Visualizer

This project is designed to help users understand sorting algorithms through visualization. Developed by Lokesh Kumar, it includes popular algorithms like Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, and Quick Sort.

The tool uses interactive bars to demonstrate how each algorithm works, with adjustable speed and array size for a customized experience.
