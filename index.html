Algorithm Hints

Bubble Sort

  • Compare adjacent elements.
  • Swap if they are in the wrong order.
  • Repeat until no swaps are needed.
  • Time: O(n²), Space: O(1).

Selection Sort

  • Find the minimum element in the unsorted portion.
  • Swap it with the first unsorted element.
  • Move the boundary of the sorted portion.
  • Time: O(n²), Space: O(1).

Insertion Sort

  • Build a sorted portion one element at a time.
  • Take the next element and insert it into the sorted portion.
  • Shift larger elements to make space.
  • Time: O(n²), Space: O(1).

Merge Sort

  • Divide the array into two halves.
  • Recursively sort each half.
  • Merge the sorted halves, comparing elements.
  • Time: O(n log n), Space: O(n).

Quick Sort

  • Choose a pivot (here, last element).
  • Partition array so elements < pivot are on the left, > pivot on the right.
  • Recursively sort the subarrays.
  • Time: O(n log n) average, O(n²) worst, Space: O(log n).
